# mtglike


## Use

Dependencies :
- gdal
- pyinotify
- xarray

Useful commands :
- to clean the folders : find automate \( -type f -o -type l \) -delete
- to start the script : python3 event_handler.py
- to create a link and start the process (on another terminal) : ln -s /path/to/file.nc automate/reception/
